Smart Shopping Cart

An ambitious attempt at create a rendition of the system implemented in Amazon Go stores which are currently only available in USA, Dec 13, 2020

The Objective: To design and implement an ‘smart’ shopping cart with its own web app

ESP32 NodeMCU is the µC that integrates with the EM18 RFID reader to read the tag off items added to the ‘cart’ it then publishes the this information to AWS IoT Core via Wifi.

Face Mask Detector

Design a model that detects whether a person is wearing a Face Mask or Not with the help of Convolutional Neural Network (CNN), Jun 14, 2020
Image

The dataset used for this project consists of 1376 images out of which 690 of these are of faces with mask, and the remaining 686 are of those without mask.

The resulting model acquired a testing accuracy of 96.376%.
